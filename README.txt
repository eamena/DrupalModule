This is a custom module that will convert an specific 
amount of money in dollars in to the value in BitCoins.
In order to get the current exchange rate it uses the App ID
from open exchange rates
http://openexchangerates.org/api/latest.json?app_id=483e2ab4893b4cd8ac43f1d9e7124b5d

This module was tested in a hosted website that can be find at:
http://digitalbullcreative.com/esau/esau/moneyconverter

The Form Tab its the main form where it asks the user for the 
APP ID and the input value being the amount of money to evaluated. 
It also handles errors by only accepting decimal values on the 
input value.

The Submissions Tab shows the values that are stored in the database
